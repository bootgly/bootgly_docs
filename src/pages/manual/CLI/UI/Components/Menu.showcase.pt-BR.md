# Menu

Os seis demos oficiais do Menu rodam ao vivo abaixo — código real do framework em PHP 8.4 WebAssembly, no seu navegador, **totalmente interativos**.

## Seleção interativa

Escolha uma variação nas abas, pressione Run e então **clique no terminal** para dar o teclado a ele: **↑/↓** (ou **←/→** no demo horizontal) movem a mira, **Espaço** alterna a seleção, **Enter** confirma.
