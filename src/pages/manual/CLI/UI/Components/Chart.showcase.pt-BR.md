# Chart

A demo oficial do Chart roda ao vivo abaixo — código real do framework em PHP 8.4 WebAssembly, no seu navegador. Use o botão de source no terminal para ler o arquivo PHP exato em execução.

## Sparkline e barras

A demo plota uma série de 15 pontos como sparkline de uma linha, e depois uma série estilo benchmark como barras horizontais rotuladas escaladas ao maior valor.

<d-block-terminal engine="bootgly-cli" title="Chart — demo ao vivo" command="demo 36" height="340">
Render de string pura — sem movimento de cursor: a mesma saída vale para terminais interativos, pipes e logs de CI. O `bootgly test benchmark` imprime esse plot de Bars após a tabela de marks.
