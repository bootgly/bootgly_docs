# Textarea

A demo oficial do Textarea roda ao vivo abaixo — código real do framework em PHP 8.4 WebAssembly, no seu navegador. Use o botão de source no terminal para ler o arquivo PHP exato em execução.

## Edição multilinha

A demo pede uma mensagem de commit. Clique no terminal e digite — Enter quebra linhas, setas navegam, Backspace mescla linhas, e Ctrl+D submete o texto.

<d-block-terminal engine="bootgly-cli" title="Textarea — demo ao vivo" command="demo 39" height="380">
A janela de linhas visíveis desliza com o cursor; um indicador esmaecido `↓ N more` conta as linhas ocultas. Em pipes, linhas do stdin são lidas até EOF — determinístico.
