# Figlet

A demo oficial do Figlet roda ao vivo abaixo — código real do framework em PHP 8.4 WebAssembly, no seu navegador. Use o botão de source no terminal para ler o arquivo PHP exato sendo executado.

## Texto grande em glifos no terminal

A demo renderiza o clássico banner do Bootgly e uma linha de dígitos — letras e números compartilham a fonte builtin `shadow`.
